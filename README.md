 Ajar-reminder
Telegram bot built with n8n to automate adhkar, fasting, Friday, and charity reminders through scheduled workflows and Telegram integration
-Ajar Reminder Bot

An automation project built using n8n and Telegram Bot API.

- Features

* Daily adhkar reminders every 12 hours.
* Monday and Thursday fasting reminders.
* Friday reminders for Surah Al-Kahf.
* Weekly charity reminders.
* Random charity opportunity selection from Ehsan platform links.
* Automated scheduling and message delivery.

- Technologies Used

* n8n
* Telegram Bot API
* JavaScript

- Workflow Overview

1. Daily Adhkar Workflow
2. Monday & Thursday Fasting Workflow
3. Friday Reminder Workflow
4. Ehsan Charity Reminder Workflow

- Purpose

The project helps users maintain beneficial habits through automated reminders delivered directly via Telegram.
